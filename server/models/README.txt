This folder contains architecture of database
Each file such as Curve.js.. CurveData.js.. Link,js.. is correspond with Curve table, CurveData table ...

Validation is used for ensuring input data is valid
It is defined in Model Declaration like Project.js( keyword:validate)
We can't define more or custom message of error
